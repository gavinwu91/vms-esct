# VMS 极客暗黑风设计规范 (VMS Design Specification)

本规范旨在定义 VMS 系统在暗黑模式下的视觉基准，确保后续业务模块开发在色彩、字体、间距和组件表现上保持高度一致。

---

## 1. 色彩系统 (Color Palette)

系统采用 **“深邃蓝-荧光青”** 为核心配色方案，强调科技感与高对比度。

### 1.1 品牌色与辅助色
| 类别 | 变量名 | 色值 (HEX/RGBA) | 用途描述 |
| :--- | :--- | :--- | :--- |
| **主色 (Primary)** | `--default-main-color` | `#0ea5e9` | 主按钮、选中态、进度条 |
| **高亮色 (Highlight)** | `--default-main-color-hight` | `#38bdf8` | 悬停态、激活态图标、标题强调 |
| **危险色 (Danger)** | `--color-danger` | `#ef4444` | 删除、报错、警示 |
| **背景基色** | `body-background` | `#020617` | 页面最底层底色 |

### 1.2 容器与面板 (Glassmorphism)
系统广泛采用半透明毛玻璃效果。
| 类别 | 色值 | 滤镜 (Blur) |
| :--- | :--- | :--- |
| **标准卡片/内容块** | `rgba(15, 23, 42, 0.4)` | `blur(10px)` |
| **弹窗/抽屉背景** | `rgba(8, 22, 45, 0.98)` | `blur(20px)` |
| **侧边栏背景** | `linear-gradient(180deg, #020617 0%, #020617 100%)` | - |
| **顶部栏背景** | `linear-gradient(90deg, #020617 0%, #0f172a 100%)` | `blur(8px)` |

---

## 2. 文字系统 (Typography)

为了适应“高密度”信息展示，系统默认字号较小，字体偏向硬朗的黑体。

### 2.1 字体家族
*   **常规文字**：`'Inter', 'Segoe UI', 'Roboto', 'PingFang SC', sans-serif`
*   **技术/ID类文字**：`'Courier New', monospace` (用于显示 ID、代码、加密等级等)

### 2.2 字号与颜色
| 级别 | 大小 | 颜色 (HEX/RGBA) | 示例 |
| :--- | :--- | :--- | :--- |
| **一级标题** | `16px` | `#f8fafc` (纯白偏蓝) | 模块名称、大弹窗标题 |
| **二级标题** | `14px` | `#38bdf8` (荧光蓝) | 卡片标题、表单分组 |
| **正文/表单标签** | `13px` | `#cbd5e1` (浅灰) | 列表内容、Label |
| **次要/占位符** | `12px` | `#64748b` (暗灰) | 提示文字、Placeholder |

---

## 3. 组件规范 (Component Standards)

### 3.1 表单输入 (Input / Select)
*   **背景色**：`rgba(15, 23, 42, 0.6)`
*   **边框**：默认 `1px solid rgba(148, 163, 184, 0.1)`，聚焦时为 `#0ea5e9`
*   **圆角**：`4px` 或 `6px`

### 3.2 表格 (Table)
*   **表头**：`rgba(15, 23, 42, 0.6)` 背景，字体颜色 `#38bdf8`
*   **行高**：默认 `40px` - `44px` (高密度布局)
*   **斑马线/分割线**：`rgba(148, 163, 184, 0.1)`

### 3.3 按钮 (Button)
*   **VMS 主按钮**：`vms-main-button` (深蓝实心)
*   **战术按钮**：`vms-track-button` (琥珀色/特定任务)
*   **取消/重置**：`transparent` 背景，带灰色或蓝色描边

## 4. 关键 CSS 变量参考

你可以直接在代码中使用这些变量来保持风格统一：

```css
/* 颜色 */
var(--el-color-primary);          /* #0ea5e9 */
var(--default-main-color);       /* #0ea5e9 */
var(--default-main-color-hight); /* #38bdf8 */

/* 字体 */
var(--el-font-size-base);        /* 13px */
var(--el-text-color-primary);    /* #f8fafc */
var(--el-text-color-regular);    /* #cbd5e1 */

/* 背景 */
var(--app-content-bg-color);     /* transparent (露出底层渐变) */
var(--el-bg-color);              /* rgba(2, 6, 23, 0.98) */
```

---

## 5. 开发建议
1.  **避免使用纯白或纯黑背景**，尽量使用 `rgba` 透明度。
2.  **内容区域务必包裹在 `ContentWrap` 组件中**，它会自动处理圆角、毛玻璃和边距。
3.  **多用图标配合文字**，图标颜色建议使用 `inherit` 或 `--default-main-color-hight`。

## 6. 样式文件与映射关系 (Style File Mapping)

为了方便开发人员快速定位和维护样式，请参考以下映射表：

| 样式文件路径 | 负责区域 / 功能 | 修改建议 |
| :--- | :--- | :--- |
| **`src/styles/var.css`** | **全局基础变量** | 修改品牌主色、全局字号、基础背景变量。 |
| **`src/styles/vms-ui.scss`** | **核心皮肤与框架样式** | 修改侧边栏菜单、顶部工具栏、全局 Element 组件（Table/Card）的重塑样式。 |
| **`src/styles/vms.scss`** | **自定义公共类** | 修改如 `.vms-main-button` 等自定义业务按钮样式。 |
| **`src/styles/index.scss`** | **全局入口** | 负责样式引入优先级管理，不建议在此直接写业务样式。 |
| **`src/components/ContentWrap`** | **通用内容容器** | 业务页面的“半透明面板”来源，修改其 `.el-card` 属性可改变全局面板质感。 |
| **`src/layout/Layout.vue`** | **主布局容器** | 修改主视图区的 `background` 或页面切换动画。 |

## 7. 常见场景开发指南

### 7.1 我想创建一个新页面，该如何起步？
1.  **容器**：在 `.vue` 文件中使用 `<ContentWrap>` 作为最外层包装。
2.  **搜索区**：利用 `<Search>` 组件，它会自动适配我们的暗黑输入框样式。
3.  **按钮**：主操作使用 `type="primary"`，次要操作建议使用 `vms-cancel-button`。

### 7.2 我想调整某个 Element 组件的全局暗黑样式？
1.  首先查看 `vms-ui.scss` 的最后一部分，那里集中处理了全局覆盖。
2.  尽量使用 `!important` 确保在 scoped 样式之上生效。
3.  优先使用 `rgba()` 而不是纯色，以保持玻璃拟态的通透感。
